Algunos de mis proyectos Open Source

  • scala-steward-action: Una "Github Action" para lanzar Scala Steward en tus propios repositorios.
  • http4s-munit: Integración entre http4s & MUnit.
  • sbt-github: Plugin SBT para leer "settings" como license o developers de GitHub.
  • sbt-fix: Plugin SBT para mejorar el dúo "scalafix" & "scalafmt".
  • sbt-modules: Plugin SBT para simplificar la creación de módulos dentro de un proyecto.
  • sbt-ci: Un set de archivos, plantillas de documentación y "workflows" de "GitHub Actions" para librerías Scala.
